# 儿童识字小报生成器

一个基于AI的儿童识字小报生成工具，帮助5-9岁儿童学习汉字。

## 功能特点

- 🎨 **AI生成图片**：使用Nano Banana Pro API生成生动的小报图片
- 📝 **智能词汇生成**：通过GLM API动态生成主题相关词汇
- 🔤 **拼音标注**：自动为每个词汇添加带声调的拼音
- 📱 **响应式设计**：适配各种屏幕尺寸
- 💾 **本地存储**：API配置安全保存在浏览器本地

## 使用步骤

### 1. 配置API

#### 图片生成API (Nano Banana Pro)
- API地址：`https://api.kie.ai`（默认）
- API密钥：从 [Nano Banana Pro](https://kie.ai/api-key) 获取

#### 词汇生成API (GLM)
- API地址：输入您的GLM API地址
- API密钥：输入您的GLM API密钥

### 2. 输入信息
- **主题/场景**：例如：超市、医院、公园、学校
- **小报标题**：例如：快乐超市、我的医院

### 3. 确认词汇
- 系统自动生成15-20个相关词汇
- 可按类别筛选查看
- 支持重新生成

### 4. 生成小报
- AI将根据词汇生成完整的小报图片
- 等待约1-3分钟

### 5. 下载图片
- 点击下载按钮保存高清图片
- 文件名格式：`识字小报_主题_标题_时间.png`

## API获取方式

### Nano Banana Pro
1. 访问 [https://kie.ai/api-key](https://kie.ai/api-key)
2. 注册账号并创建API Key
3. 复制API Key到配置页面

### GLM (智谱AI)
1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册账号并创建API Key
3. 获取API地址和密钥

## 技术架构

- **前端**：纯HTML5 + CSS3 + JavaScript
- **API调用**：Fetch API
- **存储**：localStorage
- **无需构建工具**：直接在浏览器中运行

## 本地运行

1. 克隆或下载项目
2. 使用本地服务器运行（避免跨域问题）：

```bash
# 使用Python
cd children-literacy-report
python3 -m http.server 8000

# 或使用Node.js
npx serve .
```

3. 访问 `http://localhost:8000`

## 注意事项

- API密钥仅保存在浏览器本地，不会上传到服务器
- 生成图片需要一定时间，请耐心等待
- 首次使用请先测试API连接
- 确保网络连接稳定

## 常见问题

**Q: 生成的词汇不准确？**
A: 可以点击"重新生成"按钮获取新的词汇列表。

**Q: 图片生成失败？**
A: 请检查API密钥是否正确，以及账户余额是否充足。

**Q: 无法下载图片？**
A: 尝试右键图片另存为，或检查浏览器下载设置。
