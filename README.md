# 儿童识字小报生成器

一个基于AI的儿童识字小报生成工具，帮助5-9岁儿童通过生动图片学习汉字和拼音。

## 功能特点

- 🎨 智能生成识字小报图片
- 📝 自动生成主题相关词汇（含拼音）
- 🎯 支持自定义主题和标题
- 📱 响应式设计，支持多端访问
- 💾 无需安装，直接在浏览器使用

## 使用说明

### 1. 准备工作

1. 获取 Nano Banana Pro API密钥：
   - 访问 [API密钥管理页面](https://kie.ai/api-key)
   - 注册并获取API密钥

2. 本地运行：
   ```bash
   # 使用Python
   python -m http.server 8000

   # 或使用Node.js
   npx serve .
   ```

### 2. 使用步骤

1. **输入API密钥**
   - 在页面顶部输入您的API密钥
   - 点击"保存"按钮

2. **输入主题信息**
   - 输入场景主题（如：超市、医院、公园）
   - 输入小报标题（如：快乐超市）
   - 点击"生成词汇列表"

3. **确认词汇**
   - 查看自动生成的词汇列表
   - 可以取消不需要的词汇
   - 点击"确认使用这些词汇"

4. **生成小报**
   - 等待AI生成图片（约1-3分钟）
   - 实时查看生成进度

5. **下载使用**
   - 预览生成的小报
   - 点击"下载高清图片"保存到本地

## 常见问题

### Q: 为什么生成失败？
A: 可能的原因：
- API密钥无效或余额不足
- 网络连接问题
- 输入内容包含敏感词

### Q: 如何提高生成质量？
A: 建议：
- 使用具体、明确的主题
- 标题要生动有趣
- 选择合适的词汇

### Q: 可以批量生成吗？
A: 目前不支持批量生成，需要逐个创建。
